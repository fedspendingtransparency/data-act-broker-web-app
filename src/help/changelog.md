#### November 12, 2025{section=changelog}
In this release of the Broker, we:

* Updated various aspects of the Broker to handle alphanumeric characters in assistance listing numbers. Note, no assistance listing numbers have been processed yet with the format change. Only FABS36 (which checks the assistance listing format) has been changed from a fatal error to a warning.
* Resolved a minor frontend bug on the initial DABS upload page.
