#### September 30, 2025{section=changelog}
In this release of the Broker, we:

* Per GSDM 1.1.1,
    * Added FABS 34.3 to require PeriodOfPerformanceStartDate and PeriodOfPerformanceCurrentEndDate for Grants and Cooperative Agreements.
    * Added USSGLs 480210, 497210 to Files B and C and updated several rules to use them (B6, C6, B13, C18).
* Updated Help Page side bar navigation for clarity and added the Submission Window Calendar for FY26.
